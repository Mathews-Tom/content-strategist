# Content Quality Checklist

Verification criteria applied to every content piece before delivery.

---

## Evidence Standards

- [ ] Every quantitative claim has a source or stated assumption
- [ ] No invented statistics, quotes, or attributions
- [ ] Numbers are consistent across all channel versions
- [ ] Percentages include the base ("37% of enterprise teams" not "37%")
- [ ] Date-sensitive claims include timeframe ("as of Q1 2025")

## Structure Verification

- [ ] Opening paragraph answers "why should I care?"
- [ ] Each section advances the core message
- [ ] No section repeats information from another section
- [ ] Transitions connect ideas, not just fill space
- [ ] Conclusion adds value (takeaway, CTA, framework) — not a summary of what was said

## Channel Compliance

### LinkedIn

- [ ] Under 3,000 characters
- [ ] Hook visible before "see more" fold (~210 chars)
- [ ] No more than 5 hashtags
- [ ] Standalone — makes sense without reading the blog post
- [ ] Takes a distinct angle from other channel versions

### Blog / Article

- [ ] Under 3,000 words
- [ ] Subheadings every 200-300 words
- [ ] Paragraphs under 4 sentences
- [ ] Code blocks have syntax highlighting and context
- [ ] Meta description present (150-160 chars)

### Slides

- [ ] Under 20 slides per deck
- [ ] Under 30 words per slide
- [ ] One idea per slide
- [ ] Speaker notes present for every content slide
- [ ] Visual hierarchy clear (title > subtitle > body)

### PDF Report

- [ ] Cover page with title, date, version
- [ ] TOC present if over 3 pages
- [ ] Executive summary present if over 5 pages
- [ ] Section numbering consistent
- [ ] Page numbers present

---

## Humanization Checks

### AI Pattern Detection

| Pattern | Action |
|---------|--------|
| "In today's [adjective] landscape" | Rewrite with specific context |
| "It's worth noting / important to note" | Delete, state directly |
| Sentence starting with "Furthermore" / "Moreover" / "Additionally" | Rewrite transition |
| Three or more items in "X, Y, and Z" pattern repeated | Vary sentence structure |
| Every paragraph same length | Vary paragraph length |
| "This [noun] is a [adjective] [noun] that" | Rewrite without template |
| Bullet points all starting with same word | Vary openers |

### Voice Consistency

- [ ] Tone matches the target channel (see tone-adaptation-guide.md)
- [ ] No register shifts within a single piece (formal to casual mid-paragraph)
- [ ] Contractions used consistently (all or none per piece, per channel rules)
- [ ] Active voice predominates (>80% of sentences)
- [ ] No hedging language: "might", "perhaps", "it seems"

---

## Cross-Channel Consistency

- [ ] Core message present in every piece
- [ ] No contradictory claims between channels
- [ ] Angle differs per channel (not a shortened/lengthened copy)
- [ ] Data points identical where cited across formats
- [ ] Brand voice recognizable across all pieces

---

## Final Gate

A content piece ships when ALL applicable checks pass. If a check fails, fix it before delivery — do not flag it as a known issue and ship anyway.
